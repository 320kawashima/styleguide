###単語の省略について

以下の単語については省略  
- banner → bnr  
- content → cont  
- description → desc  
- navigation → nav  
- section → sec  


##Utility

ユーティリティの命名はEmmetのチートシートを参考にすること。  
例えば、font-weight: bold;を書けたければ、プレフィックス以降は「:」を省略したfwbをつけ.u-fwbとする。  
!importantをつける。
